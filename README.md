# Betriebssystem projekt
Dieses Projekt soll zur vertiefung von der Funktionsweise des Speichers helfen, hierfür muss eine Virtuelle Block basierte Festplatte erzeugt werden und mit einem INode oder FAT Dateisystem belegt werden.zusätzlich soll die Nutzung über ein Interface ermöglich werden.
